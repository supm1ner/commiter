# 🟩 commiter

> спамит коммиты. заполняет граф. выглядит продуктивно.

![GitHub commit activity](https://img.shields.io/github/commit-activity/m/supm1ner/commiter)

## что это

программа которая делает коммиты сама в себя. бесконечно. быстро. много.
граф активности на гитхабе будет зелёным навсегда.

## запуск

```bash
git clone https://github.com/supm1ner/commiter
cd commiter

# бесконечный режим
python committer.py

# 500 коммитов без задержки
python committer.py -n 500 -d 0

# медленно и вдумчиво
python committer.py -d 1 -b 5
```

## флаги

| флаг | описание | default |
|------|----------|---------|
| `-n` | количество коммитов (0 = ∞) | 0 |
| `-d` | задержка между коммитами (сек) | 0.1 |
| `-b` | пушить каждые N коммитов | 10 |

## требования

- python 3.x
- git
- настроенный remote (уже есть если клонировал)
